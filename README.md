# Змейка (Snake Game)

Это учебное приложение, реализующее классическую игру «Змейка».
Проект разработан с использованием библиотеки Pygame и предназначен
для запуска на операционной системе Windows.

## Описание проекта

Приложение представляет собой однопользовательскую игру, в которой
игрок управляет змейкой, перемещающейся по игровому полю.
Цель игры — собирать еду, увеличивая длину змейки, и избегать
столкновений со стенами и собственным телом.

## Основные возможности

* Управление змейкой с клавиатуры.
* Рост змейки при поедании еды.
* Генерация еды в случайных точках игрового поля.
* Обработка столкновений:
  * со стенами игрового поля;
  * с собственным телом змейки.
* Завершение игры при столкновении.
* Отображение игрового процесса в реальном времени.

## Технологический стек

* Язык программирования: Python
* Графическая библиотека: Pygame
* Среда разработки: PyCharm
* Платформа: Windows
